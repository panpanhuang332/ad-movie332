# 微電影廣告 · 一週學會拍攝

一個以手機雙機拍攝為基礎的微電影廣告製作教學網站，涵蓋導演、攝影、燈光、收音四大核心技術，以及相對應的器材選購指南。

## 🎬 線上預覽

啟用 GitHub Pages 後，直接訪問：  
`https://你的帳號.github.io/這個repo名稱/`

## 🚀 如何部署到 GitHub Pages

1. 在 GitHub 建立一個新的 public repository
2. 把整個資料夾的內容上傳（或 `git push`）
3. 進入 repo 的 **Settings → Pages**
4. **Source** 選「Deploy from a branch」，Branch 選「main」，資料夾選「/ (root)」
5. 儲存後等幾分鐘，網站就會出現在 `https://你的帳號.github.io/repo名稱/`

## 📚 內容章節

- **導演**：鏡頭語言、攝影機運動（含四支示範影片）、場面調度、視覺風格
- **攝影**：曝光三要素、四個構圖法則（含示意圖）、景深控制、LOG 格式
- **燈光**：自然光運用、三點布光、色溫與情緒、進階光效（林布蘭／分割／RGB）
- **收音**：聲音四層次、麥克風種類、監聽與混音
- **器材指南**：攝影、燈光、收音設備的選購與租借建議
- **設備比較**：手機雙機（S22+S23, iPhone 16+17）與各級專業攝影機的對應分析
